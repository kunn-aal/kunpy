1. Lists[]
    - mutuable in place (insert, remove, pop, updatable)
    - repetition is allowed
    - heterogenous data is permitted (different data types allowed in one list)

2. Tuple()
    - immutable (nothing can be changed)
    - repetition is allowed
    - heterogenous (only one data type is allowed)

3. Sets{}

    - repeition is not allowed
    - unordered
